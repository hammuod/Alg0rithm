# Contributing to Alg0rithm

Welcome. We are a team of dedicated Arab developers who believe in simplicity, ethics, and technical perfection. To maintain the integrity of this project, every contributor must strictly follow this Constitution.

---

## The Golden Rule
"You are free as long as you do no harm."
We grant you the freedom to learn and develop, but harming the project’s identity, code quality, or ownership rights is strictly prohibited.

---

## 1. Technical Standards (No Bloat Policy)
* Vanilla Only: No external libraries allowed (No jQuery, No React, No Bootstrap). The code must be 100% self-built.
* Unified Structure: You must follow the defined folder hierarchy (/css, /js, /docs).
* Visual Identity: Do not change the unified font, colors, or CSS variables. The design must remain consistent and minimal.
* Performance: Any code that adds unnecessary weight (Bloat) without a vital function will be rejected immediately.

## 2. The Mastery System (5-Times Review)
We do not push "good enough" code. Before submitting a Pull Request (PR):
1. Review 1: Does it work?
2. Review 2: Is there any bloat?
3. Review 3: Is the structure correct?
4. Review 4: Is it cyber-secure?
5. Review 5: Is it simple enough for a student to understand?
Every single line is reviewed 5 times before merging.

## 3. SEO and Indexing
* The sitemap.xml must be updated with every major update.
* The robots.txt file must not be modified without the Lead Developer's approval.
* Use Semantic HTML to maintain the highest SEO score.

## 4. Team Ethics and Rotation
* The project operates on a weekly rotation system among 5 developers.
* Each developer is responsible for daily updates during their assigned week.
* Arab Ethics: Honesty in coding, integrity in documentation, and mutual respect are our foundations.

## 5. Copyright and Integrity
* This project is protected under the AGPL-3.0 License.
* Stealing code or claiming ownership of this work is strictly forbidden.
* Any derivative work must credit the original creator (Hammoud - Alg0rithm).
* Our history of 77+ deployments serves as a permanent record of ownership.
